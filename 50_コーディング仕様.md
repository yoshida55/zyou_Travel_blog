# ブログサイト コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| コンテンツ幅 | 1200px（横パディング16px） |
| 全幅背景 | グローバルナビゲーション、フッター |
| フォント | Google Fonts「Noto Sans」「Noto Sans JP」 |
| レイアウト | 2カラム（メインエリア + サイドバー） |

---

## 各セクション仕様

### Header

- **ヘッダー全体**
  - PC時、スマホ時ともに固定（`position: fixed`）
  - ロゴの下に全幅のグローバルナビゲーションを配置

- **ロゴ**
  - `h1`タグで記述

- **グローバルナビゲーション**
  - `nav`タグは全幅で背景カラーを設定
  - メニューは最大幅1200pxを指定して中央に配置
  - `ul`、`li`タグで横並びに揃える

---

### Pickup（ピックアップエリア）

- **エリア全体**
  - `div`タグで囲む（`id="pickup"`）
  - Flexboxで3つの記事を横並びに配置

- **各記事**
  - `article`タグで囲む
  - タイトル部分は`h2`タグで記述
  - リンクは`:hover`で下線を消す（`text-decoration: none`）

---

### Container（メインエリア + サイドバー）

- **コンテナ全体**
  - `div`タグで囲む（`id="container"`）
  - Flexboxで`main`と`aside`を横並びに配置
  - それぞれのコンテンツの横幅は可変（レスポンシブ対応）

---

### Main（メインエリア）

- **エリア全体**
  - `main`タグで囲む
  - コンテンツのメインとなる記事一覧を配置

- **各記事**
  - `article`タグで囲む
  - タイトルは`h2`タグで記述
  - 更新日とカテゴリ名は`ul`、`li`タグを使用してFlexboxで横並びに配置
  - 日付は`<time>`タグを使用

- **画像**
  - 記事のサムネイル画像を配置

- **本文抜粋**
  - 記事の概要テキストを表示

- **READ MOREリンク**
  - 各記事の詳細へのリンク

---

### Aside（サイドバー）

- **エリア全体**
  - `aside`タグで囲む
  - サイドバー全体を配置

#### Authorエリア

- `section`タグで囲む
- 名前は`h3`タグで記述
- プロフィール画像に`border-radius`を指定して丸く切り抜く
- `width`と`height`で円のサイズを調整

#### Rankingエリア

- `section`タグで囲む
- エリアのタイトルは`h3`タグ
- 各記事は`article`タグで囲む
- 記事タイトルは`h4`タグで記述
- ランキング上位3記事を表示

#### Archiveエリア

- `section`タグで囲む
- エリアのタイトルは`h3`タグ
- アーカイブリストは`ul`、`li`タグで記述
- 年月と記事数を表示

---

### Footer

- **フッター全体**
  - `footer`タグで囲み、背景カラーを設定
  - 全幅の背景

- **コンテンツ配置**
  - About、Menu、Twitterの3つのブロックを横並びに配置
  - 各コンテンツを`section`タグで囲む
  - 外側の`div`タグでFlexboxを指定して横並びに配置
  - 各セクションのタイトルは`h3`タグで記述

#### Aboutエリア

- 概要テキスト
- リンクリスト（プロフィール詳細、お仕事の依頼、お問い合わせ）
- リンク先頭の黒矢印は`a`タグの`::before`で指定
- 擬似要素でCSSのみで矢印を作成

#### Menuエリア

- ナビゲーションメニューのリスト
- `ul`、`li`タグで記述

#### Twitterエリア

- Twitterタイムラインの埋め込み
- 公式サイトから埋め込み用コードを生成して使用

---

## レスポンシブ対応

```css
/* PC表示 */
@media (min-width: XXXpx) {
  /* PC用スタイル */
  /* ピックアップ記事：3カラム横並び */
  /* メインエリアとサイドバー：2カラム横並び */
  /* フッター：3ブロック横並び */
}

/* スマホ表示 */
@media (max-width: XXXpx) {
  /* スマホ用スタイル */
  /* ピックアップ記事：縦並び */
  /* メインエリアとサイドバー：縦並び（メインが上、サイドバーが下） */
  /* フッター：縦並び */
}
```

**注意点**
- ブレイクポイントはデザインカンプで確認してください
- メインエリアとサイドバーの幅は可変で設定
- スマホ時の並び順に注意（必要に応じて`order`プロパティで調整）

## 使用する主なHTMLタグ

| タグ | 用途 |
|------|------|
| `h1` | ヘッダーロゴ |
| `h2` | ピックアップ記事タイトル、メイン記事タイトル |
| `h3` | サイドバーセクションタイトル、フッターセクションタイトル |
| `h4` | ランキング記事タイトル |
| `nav` | グローバルナビゲーション |
| `article` | 各記事（ピックアップ、メイン、ランキング） |
| `section` | サイドバー各エリア、フッター各エリア |
| `main` | メインコンテンツエリア |
| `aside` | サイドバーエリア |
| `footer` | フッターエリア |
| `time` | 日付表示 |

---

## 使用する主なCSSプロパティ・テクニック

| プロパティ/テクニック | 用途 |
|----------------------|------|
| `position: fixed` | ヘッダー固定 |
| `display: flex` | 横並びレイアウト（ナビ、ピックアップ、メイン+サイドバー、フッター） |
| `flex-direction: column` | スマホ時の縦並び |
| `max-width` | コンテンツ幅制限 |
| `padding` | 左右余白 |
| `border-radius` | プロフィール画像の円形切り抜き |
| `::before` | 矢印アイコン（擬似要素） |
| `:hover` | リンクホバー時の下線削除 |
| `text-decoration` | 下線の制御 |
